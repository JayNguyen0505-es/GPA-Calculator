# GPA-Calculator
My motivation for building this small project came from the difficulty of calculating my GPA manually. I often had to write down my subjects, ECTS, and grades, then use a calculator to compute the result, which was slow and time-consuming. This repeated process made it inconvenient whenever I wanted to quickly check my academic performance.

To solve this, I decided to create a simple GPA calculator using CodeSandbox. This project allows users to input their subjects, ECTS, and grades on a 20-point scale, and instantly compute their weighted average.
